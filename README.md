# Projeto IREDE

Esse repositório foi feito com o objetivo de armazenar os códigos produzidos na etapa de desenvolvimento da aplicação API REST em Node.js e orientar adequadamente como executar a aplicação feita por meio de instruções. E, além disso, ele servirá como um guia de informações necessárias tanto para a execução da API quanto informações relacionadas as componentes utilizados e demais ferramentas de desenvolvimento.
